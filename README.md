## What it does

AutoTyper simulates realistic human keystrokes in any text field — Google Docs, Microsoft Word, online forms, anywhere. It handles the pacing, typos, pauses, and paragraph rhythm so it looks natural.

### Paste Mode
Paste any text, set how long you want it to take (1–90 minutes), click Start. You have 8 seconds to switch to your document before typing begins.

### AI Mode
Describe what you want written — AutoTyper sends your prompt to an AI, streams the response back in real time, and then types it into your document at human speed.

---

## How to use

1. **Install** — run `AutoTyper Setup 1.0.0.exe` and follow the prompts
2. **Open your document** — Google Docs, Word, anything with a text field
3. **Launch AutoTyper** from your Desktop or Start Menu
4. Choose **Paste Mode** or **AI Mode**
5. Set your text and duration, then click **▶ Start**
6. Switch to your document within **8 seconds** — typing begins automatically

## FAQ

**Does it work in Google Docs?**
Yes. Click inside the document before the 8-second countdown ends.

**Will it type in the wrong window?**
Only if you don't switch windows during the countdown. AutoTyper types into whatever is focused when the timer hits zero.

**Is the AI free?**
Yes — it uses the Groq API which is free for normal usage.

**Windows Defender shows a warning on install — is it safe?**
Yes. The warning appears because the app isn't code-signed (that costs money). Click **More info → Run anyway** to proceed.
